# Описание LATCH

### Что такое Latch?
**Latch (защёлка)** - это одна из простейших последовательностных схем, которая хранит один бит информации (0 или 1). Главная особенность элемента — он является **асинхронным** и **уровнечувствительным**. Это значит, что его выход меняется сразу же, как только меняется вход, при условии, что управляющий сигнал (Enable) активен.

---

### RS-триггер
**RS-триггер** - простейший пример защёлки, имеет два входа **R** и **S** (**Reset** и **Set** соответственно) и два выхода **Q** и **$\overline{Q}$**. Состоит из двух перекрёстно включённых элементов **ИЛИ-НЕ**.
